## Features

- Photo Capture with Real-Time Filters  
- Video Recording with Audio Support  
- Front & Back Camera Switching (Mobile Supported)  
- Live Filter Effects (Canvas Rendering)  
- Recording Timer with Pause/Resume  
- Instant Download (Photo & Video)  
- Fully Responsive & Mobile Compatible  
- Secure Media Access (HTTPS Required)

- HTML5

---

## How It Works

SnapWeb captures live camera input using `getUserMedia`, applies real-time visual filters through the Canvas API, and records media using the MediaRecorder API with automatic MIME type fallback for maximum browser compatibility.

No frameworks. No dependencies. Just pure web technology.
